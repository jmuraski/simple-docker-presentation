What is Docker

  • Docker Engine
    • Run Docker Containers
    • Build Docker Containers
  • Docker Hub
    • Store Docker Containers
    • Share Docker Containers
  • Docker Machine
    • Install Docker Engine on hosts
  • Docker Compose
    • Define and compose multiple containers
    • Single host
  • Docker Swarm
    • Define and compose multiple containers
    • Multiple hosts
